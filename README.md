

# ⛳ Golf Charity Platform

A modern web application designed to connect golf enthusiasts with charitable initiatives through interactive campaigns, donations, and community engagement.

---

## 📌 Overview

The Golf Charity Platform enables users to participate in charity-driven golf events, track contributions, and engage with a community focused on social impact.

Built with a focus on scalability, performance, and clean user experience, the platform demonstrates modern frontend development practices and structured project architecture.

---

## ✨ Features

- 🎯 Create and explore charity campaigns  
- 💰 Donation-ready platform (integration-ready)  
- 📊 Track participation and contributions  
- 🧑‍🤝‍🧑 Community-driven engagement  
- 📱 Fully responsive and modern UI  

---

## 🛠️ Tech Stack

- **Frontend:** React + Vite  
- **Styling:** Tailwind CSS  
- **Backend & Database:** Supabase  
- **Testing:** Playwright / Vitest
